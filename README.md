# Quingo Main Service

Основной сервис платформы Quingo. Реализует бизнес-логику для манипуляциями квизами, упралвяет проведением игр. 


## 🛠 Технологический стек

| Технология           | Роль в проекте                                                |
|:---------------------|:-------------------------------------------------------------|
| **Kotlin**           | Основной язык программирования                               |
| **Quarkus 3.35.2**   | Kubernetes-native фреймворк (Gradle Kotlin DSL)             |
| **Kubernetes (k3s)** | Оркестрация контейнеров в кластере                          |
